# 🦷 Cleardent - Paneles de Gestión

Paneles interactivos para la gestión de contenido de las clínicas dentales Cleardent.

## 📋 Contenido

### 📅 Calendario GBP 2026
Calendario editorial completo para Google Business Profile con planificación de contenido para Marzo y Abril 2026.

**Características:**
- Planificación de Implantes Dentales (Marzo)
- Planificación de Ortodoncia Invisible (Abril)
- Filtros por tipo de contenido (POST, GALERÍA, Vídeo)
- Especificaciones detalladas de cada publicación
- Descripciones para herramientas IA con función de copiado

### 🏥 Panel de Clínicas
Sistema de gestión de imágenes para las clínicas con dos vistas:

**Panel Clínicas:**
- Instalaciones
- Equipo
- Tratamientos
- Eventos

**Panel Marketing:**
- Revisión de fotos
- Google Business Profile (GBP)
- Campañas
- Diseño
- Contenido IA

### 📋 Panel con Ejemplos
Versión mejorada del panel de clínicas que incluye ejemplos de nomenclatura de archivos para cada categoría.

## 🚀 Uso

Simplemente abre cualquiera de los archivos HTML en tu navegador para comenzar a usar los paneles.
